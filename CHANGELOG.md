# Changelog

## [0.1.0] — 2026-07-16

### Added

- Initial release.
- 256 C# and Unity problem-solving modules with expand/collapse cards.
- Keyword search, difficulty/type/tag filters, bookmarking, and deep linking.
- Dark-mode glassmorphism UI, zero external dependencies (except Google Fonts).
- Static web app — open `index.html` in any browser to use.

### Notes

- The dataset is synthetically generated as a structural scaffold.
- All answers, explanations, and code examples across all 256 modules use boilerplate placeholder text.
- The current UI caps displayed results at 100 cards — modules 101–256 are not reachable through browsing.
- This is a pre-release version. Data quality improvements are planned for future releases.
